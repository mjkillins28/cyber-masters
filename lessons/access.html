Thi video will show you how to access folders in a different PC. This skill can come in handy when you’re constantly switching computers. Log into your windows workstation, and click on the start menu. Click on computer and then click on Map Network Drive located at the top bar of the screen. A window should pop up and in the drive drop down list, you can select any available drive letter. In the folder, enter the following: 1. Substitute server-address with the server address given to you by the IT support 2. Substitute share-name with the share name given to you by IT support 3. Be sure to add the \\ at the start of the folder name and a single \ between the server and share name. If you’re going to use the map regularly, make sure to check the Reconnect at sign-in box. Once you check this option, it will automatically load the files for you every time you sign into this computer. Click finish. If all is working windows should display a box with the shared folder contents. You can then access the shared folder contents from any application by using the drive letter selected above.
